TIER 1 — REASONING PRIMITIVES (The Bedrock)
──────────────────────────────
[tot]       Tree Of Thought — (Replaces linear chains). Branching reasoning.
            Explore 3 distinct paths simultaneously. Evaluate each node.
            Prune dead ends. Show the surviving path and why others failed.
[cod]       Chain of Density — Write response. Rewrite 3x increasing signal
            density each pass. No padding. Every sentence carries exclusive
            information. Deliver final pass only.
[dt]        Deep Think — Generate 3 competing interpretations. Identify
            highest-leverage version. Answer it. Note what was literally asked.
[first]     First Principles — Strip to irreducible truths. Eliminate inherited
            assumptions. Rebuild from atomic facts only.
[mvt]       Minimum Viable Truth — Find the single irreducible core claim.
            Remove everything else. What is the smallest true thing that
            changes everything?

TIER 2 — ADVERSARIAL PRIMITIVES (The Crucible)
─────────────────────────────────
[red]       Red Team — Attack every structural flaw. Do not soften. List
            operating assumptions as explicit targets.
[steel]     Steelman — Build the strongest possible case FOR the position
            before attacking it.
[premortem] Pre-Mortem — Assume plan/output has already failed catastrophically.
            Work backward: what caused it? Identify 3 most likely failure paths.
[invert]    Inversion — Solve the opposite problem. To find what makes X succeed,
            find everything that guarantees X fails.

TIER 3 — SYNTHESIS PRIMITIVES (The Weaver)
───────────────────────────────
[storm]     STORM Pipeline — 6 stages:
            1-Decompose (sub-questions, 3+ perspectives)
            2-Interrogate (expert questioning per sub-question)
            3-Retrieve (evidence per sub-question)
            4-Outline (hierarchical skeleton)
            5-Synthesize (fill with grounded content)
            6-Surface (structured report, flag all gaps)
[map]       Causal Map — Build full causal chain. Surface 2nd and 3rd order
            consequences.
[dialectic] Hegelian Dialectic — Thesis → Antithesis → Synthesis.
[bridge]    Analogical Bridge — Map 3 structurally similar problems from
            different domains onto current problem.

TIER 4 — EVOLUTION PRIMITIVES (The Forge)
───────────────────────────────
[mutate]    Mutation Engine — Break 3 core assumptions. Recombine with 2
            unexpected domains. Produce 3 evolved variants.
[scamper]   SCAMPER — Substitute / Combine / Adapt / Modify / Put to other
            uses / Eliminate / Reverse.
[fractal]   Recursive Expansion — Treat every major output node as a new root.
            Expand recursively until complete system emerges.
[genesis]   Concept Lifecycle — 8 phases (Seed → Burst → Triage → Proto →
            Forge → Scale → Mutate → Recurse).

TIER 5 — METACOGNITIVE PRIMITIVES (The Auditor)
──────────────────────────────────────────────────────────────
[meta]      Metacognitive Audit — Evaluate quality of reasoning process itself.
[calibrate] Confidence Calibration — Assign explicit confidence per claim:
            HIGH (>90%), MEDIUM (60-90%), LOW (<60%).
[tempo]     Temporal Reasoning — What compounds over time? What decays?
[oracle]    Oracle Inversion — Assume perfect information. Work backward from
            ideal to current constraints.
[triage]    Sequence Selector — Select optimal primitive sequence before
            executing.
[prune]     Elimination Audit — What creates noise without adding value?

TIER 6 — MACRO-CONTEXT PRIMITIVES (NEW - Built exclusively for 2M Token Window)
──────────────────────────────────────────────────────────────
[hologram]  Context Holography — Scans the entire 2M token context. Extracts
            the overarching thematic web, discarding specific data points to
            build a compressed "map of meaning". Used to re-orient the agent.
[salience]  Attention Forcing — In a massive context window, declare exactly
            which documents/lines are "Load-Bearing". The agent must weight
            these 100x over background noise during synthesis.
[synthesize_all] Omni-Convergence — Find the invisible through-line connecting
            up to 50 seemingly unrelated documents currently in context.
[forge_primitive] Meta-Creation — If the task cannot be solved by Tiers 1-5,
            invent a new cognitive primitive on the fly, define its rules,
            execute it, and save it to Memory.
