# SOURCE SELECTION MATRIX

Status: Proposed control document
Purpose: Rank repo sources by current usefulness for Layer 2 truth, Layer 4 artifacts, and deferred analysis.

## Selection standard

Use a source now if it improves one of these immediately:
- trustworthy profile truth
- trustworthy task or obligation state
- stable workflow or system rules
- preserved artifact canon

Defer a source if it is mostly:
- brainstorming sprawl
- duplicate passover material
- architecture repetition
- speculative upgrades not needed for the current phase
