## RECOMMENDATION: JSON vs YAML for AI Transfer

**JSON Advantages:**
- Universal parsing across all AI systems
- Strict schema validation (fewer errors)

- Best for: Complex hierarchies, programming integration

**YAML Advantages:**
- More human-readable (if Gemini needs review)
- Cleaner list handling (indentation-based)
- Comments allowed (#)
- Best for: Configuration files, quick reference

**Recommendation:** Start with **JSON** for Gemini (most reliable), then offer YAML as backup for readability.
